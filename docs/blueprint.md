# **App Name**: BeautyFlow

## Core Features:

- Client Management: Add and manage client details including name and phone number.
- Service Tracking: Record each beauty service provided, including price, notes, and follow-up validity period (months/weeks).
- Renewal Reminders: Automated alerts for services nearing their validity date, displayed prominently on the dashboard.
- Automated WhatsApp Messages: Generate pre-filled WhatsApp messages to remind clients about service renewals, including personalized offers.
- Dashboard Analytics: Visualize key performance indicators, with a line chart of total revenue, a bar graph of new client acquisition, and a pie chart showing popular services + Hide revenue data Function, as bank apps.
- Service Registry: Centralized catalog of the professional's available services.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke tranquility and beauty.
- Background color: Very light grayish-lavender (#F5F5FF) to provide a clean and calming backdrop.
- Accent color: Muted rose (#E0BBE4) for interactive elements and highlights.
- Body and headline font: 'Alegreya', a humanist serif with an elegant and intellectual feel.
- Elegant, minimalist icons that represent beauty services and dashboard metrics.
- Clean and intuitive layout that puts key performance indicators and renewal alerts front and center.
- Subtle transitions and animations for a polished user experience.
